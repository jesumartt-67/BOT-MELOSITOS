Objetivo:
Desarrollar una aplicacion WEB, que use la Arquitectura de MicroServicios, la cual permita leer desde un PDF informacion sobre costos educativos, jornadas extensivas, costos de uniformes, fechas importantes, actividades
extra curriculares...etc que se van a almacenar en una BDs vectorial, para luego ser usada por un chatbot para responder a consultas de clientes.

Requerimientos:
1. Realizar un Front-End basico el cual permita la interaccion del chatbot con el usuario final.
2. Realizar el Back-End donde vamos a almacenar la informacion de los PDFs en una BD vectorial. 
3. Realizar una API REST-ful que interactue entre el Front-End (ChatBot) y el Back-End (BD vectorial).
